8.2 Escriba una instrucción o un conjunto de instrucciones en JavaScript para realizar cada una de las siguientes tareas:

a) Sumar los enteros impares entre 1 y 99. Utilice una instrucción for. Suponga que se han declarado las variables suma y cuenta.
b) Calcular el valor de 2.5 elevado a la potencia de 3. Use el método pow.
c) Imprimir los enteros del 1 al 20 mediante el uso de un ciclo while y la variable contador x. Suponga que la variable x se ha declarado, pero no se ha inicializado. Imprima solamente cinco enteros por línea. [Sugerencia: use el cálculo x % 5. Cuando el valor de esta expresión sea 0, inicie un nuevo párrafo en el documento de HTML5].
d) Repita el ejercicio 8.2, (c), pero ahora use una instrucción for.

a) suma = 0; for ( cuenta = 1; cuenta <= 99; cuenta += 2 ) suma += cuenta;
b) Math.pow( 2.5, 3 );
c) x = 1; document.writeln( "<p>" ); while ( x <= 20 ) { document.write( x + " " ); if ( x % 5 == 0 ) document.write( "</p><p>" ); ++x; }
d) document.writeln( "<p>" ); for ( x = 1; x <= 20; x++ ) { document.write( x + " " ); if ( x % 5 == 0 ) document.write( "</p><p>" ); }
