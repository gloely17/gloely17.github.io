7.7 Identifique y corrija los errores en cada uno de los siguientes segmentos de código:
a) while ( c <= 5 ) { producto *= c; ++c;
b) if ( genero == 1 ) document.writeln( "Mujer" ); else; document.writeln( "Hombre" );

a) Error: falta la llave derecha de cierre del cuerpo de la instrucción while. Corrección: Agregar una llave derecha de cierre después de la instrucción ++c;.
b) Error: El ; después de else produce un error lógico. La segunda instrucción de salida siempre se ejecutará. Corrección: Quitar el punto y coma después de else.
